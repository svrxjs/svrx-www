![](/assets/images/banner.png)


The [homepage](https://svrxjs.github.io/svrx-www/) of svrx.

### Development

 - Install [Jekyll](https://jekyllrb.com/docs/installation/)
 - `bundle install`
 - `bundle exec jekyll serve`
 - Open http://127.0.0.1:4000
